# Escalations Humanas

Briefings preparados pelos auditores externos quando um caso-limite exige humano real. Mesmo com pareceres dos três agentes em PASS, estes casos mantêm fail-closed até decisão explícita do usuário e registro em ADR quando houver aceite de risco.

## Casos-Limite

- Auditoria CGCRE agendada.
- Processo judicial aberto.
- Incidente LGPD com dados vazados.
- Acidente metrológico.
- Reclamação formal em órgão regulador.

## Regra

Cada briefing deve indicar evidência, especialista humano recomendado, impacto se não contratar e estado operacional da emissão.
