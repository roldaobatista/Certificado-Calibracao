# Regulator Decisions

Precedentes de interpretação normativa consolidada ficam neste diretório.

Uma decisão nova em área blocker deve registrar:

- passagem A com contexto de spec, AC e código atual;
- passagem B com pacote normativo anexado explicitamente;
- resultado comparado das duas passagens;
- self-consistency 3x quando não houver precedente;
- escalonamento D8 se houver divergência;
- owner e data da consolidação.

Aplicações futuras do mesmo caso devem consultar o precedente em vez de reinterpretar a norma do zero.
