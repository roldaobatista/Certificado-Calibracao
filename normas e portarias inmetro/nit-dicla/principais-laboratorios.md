# Principais NIT-DICLA para Laboratórios

> **Contexto:** As NIT-DICLA foram elaboradas pela antiga Divisão de Acreditação de Laboratórios (Dicla) do Inmetro. Com reorganizações, muitas migraram para o formato **DOQ-CGCRE** ou foram consolidadas. **Sempre verificar vigência e substituições no portal Cgcre.**

## Tabela de NITs mais referenciadas

| Código | Tema | Observação |
|--------|------|------------|
| NIT-DICLA-016 | Tratamento de reclamações e apelações | Verificar se foi substituída por DOQ |
| NIT-DICLA-021 | Expressão da incerteza de medição | Complementa o DOQ-CGCRE-008 |
| NIT-DICLA-026 | Auditoria interna em laboratórios | Aplicação prática do item 8.8 da ISO 17025 |
| NIT-DICLA-031 | Laboratórios em campo / móveis | Aplicações fora da sede |
| NIT-DICLA-035 | Rastreabilidade metrológica | Base para cadeia de rastreabilidade |
| NIT-DICLA-038 | Uso de materiais de referência | Critérios para MR e MRC |
| NIT-DICLA-083 | Laboratórios clínicos (ISO 15189) | Área específica |

## Uso prático

- **NITs ainda vigentes** complementam os DOQs quando detalham um aspecto específico.
- **NITs substituídas** devem ser descartadas — usar o DOQ equivalente.
- Sempre procurar no portal Cgcre se há uma orientação mais recente sobre o mesmo tema.

## Como verificar status

1. Acessar https://www.gov.br/inmetro/pt-br/assuntos/acreditacao/documentos
2. Buscar por "NIT-DICLA-XXX" ou tema correspondente.
3. Verificar data de última publicação e eventuais notas de substituição.

## Como anexar uma NIT nesta pasta

1. Baixar o PDF da versão vigente.
2. Salvar como `nit-dicla-<numero>-rev<n>.pdf`.
3. Criar ficha `nit-dicla-<numero>.md` com:
   - Número e revisão
   - Data
   - Objeto
   - Aplicabilidade
   - Se foi substituída por DOQ, anotar o equivalente
