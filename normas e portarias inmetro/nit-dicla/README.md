# NIT-DICLA — Normas para a Área de Laboratórios

As **NIT-DICLA** são normas internas do Inmetro publicadas pela Divisão de Acreditação de Laboratórios (Dicla, hoje integrada à Cgcre) que estabelecem requisitos específicos para laboratórios.

> Em razão de reorganizações internas do Inmetro, parte das NIT-DICLA migrou para o formato **DOQ-CGCRE**. Sempre verifique vigência e equivalência na fonte oficial.

## Onde consultar
- **Cgcre — documentos:** https://www.gov.br/inmetro/pt-br/assuntos/acreditacao/documentos

## NIT-DICLA mais referenciadas

| Código | Tema |
|--------|------|
| NIT-DICLA-016 | Tratamento de reclamações e apelações |
| NIT-DICLA-021 | Expressão da incerteza de medição |
| NIT-DICLA-026 | Auditoria interna em laboratórios |
| NIT-DICLA-031 | Requisitos específicos para laboratórios em campo / móveis |
| NIT-DICLA-035 | Rastreabilidade metrológica |
| NIT-DICLA-038 | Critérios para uso de materiais de referência |
| NIT-DICLA-083 | Critérios específicos para laboratórios clínicos |

## Como anexar
1. Baixar o PDF da versão vigente.
2. Salvar como `nit-dicla-<numero>-rev<n>.pdf`.
3. Atualizar índice e ficha resumo.

## Cuidados
- Antes de aplicar uma NIT-DICLA, confirme se ela **continua vigente** (não foi revogada nem substituída por DOQ-CGCRE).
- Quando aplicável, prefira o DOQ-CGCRE equivalente.
