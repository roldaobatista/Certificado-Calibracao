# DOQ-CGCRE — Documentos Orientativos da Cgcre

Os **DOQ-CGCRE** são documentos publicados pela **Coordenação Geral de Acreditação** (Cgcre/Inmetro) para orientar laboratórios, organismos de inspeção/certificação e demais partes interessadas no processo de acreditação.

## Natureza
- Têm caráter **orientativo e/ou normativo** dependendo do conteúdo.
- Detalham **como o Inmetro/Cgcre interpreta e aplica** as normas internacionais (ISO/IEC 17025, 17020, 17021, 17024, 17043, 17034, 15189) no contexto brasileiro.
- Devem ser observados por organizações em processo de acreditação ou já acreditadas.

## Onde consultar
- **Lista oficial e versões vigentes:** https://www.gov.br/inmetro/pt-br/assuntos/acreditacao/documentos
- Sempre verificar a **revisão atual** antes de aplicar.

## Principais DOQ-CGCRE para laboratórios de calibração e ensaio

| Código | Tema |
|--------|------|
| DOQ-CGCRE-001 | Critérios para acreditação de laboratórios — documento principal |
| DOQ-CGCRE-002 | Tabela de equivalência terminológica |
| DOQ-CGCRE-008 | Orientação para expressão da incerteza de medição em calibração |
| DOQ-CGCRE-019 | Orientação sobre validação de métodos analíticos |
| DOQ-CGCRE-020 | Critérios para participação em ensaios de proficiência |
| DOQ-CGCRE-026 | Acreditação de laboratórios para ensaios qualitativos |
| DOQ-CGCRE-028 | Identificação visual / uso da marca Cgcre |

## Outros DOQ-CGCRE relevantes (por área)

### Inspeção (ISO/IEC 17020)
- Documentos da série DOQ-CGCRE-OIA-* relacionados à acreditação de OIAs.

### Certificação (ISO/IEC 17021, 17065, 17024)
- DOQs específicos para organismos certificadores.

### Provedores de PT (ISO/IEC 17043)
- Documentos sobre estatística, planejamento de rodadas e tratamento de resultados.

### Produtores de MR (ISO 17034)
- Documentos sobre certificação de materiais de referência.

## Como anexar um DOQ neste repositório
1. Baixar o PDF na versão **vigente** do site da Cgcre.
2. Salvar como `doq-cgcre-<numero>-rev<n>.pdf`.
3. Atualizar a tabela acima com a data e revisão baixada.
4. Criar uma ficha de mudanças (`doq-cgcre-<numero>-changelog.md`) ao migrar para nova revisão.

## Política de revisão
A Cgcre revisa os DOQ-CGCRE periodicamente, frequentemente em decorrência de:
- Atualização de normas ISO/IEC.
- Atualização de documentos ILAC (P-series, G-series).
- Melhoria contínua do processo de acreditação.

> **Recomendação:** monitore mensalmente a página de documentos da Cgcre e estabeleça processo formal de atualização (ver `04-templates/` da pasta `iso 17025/`).
