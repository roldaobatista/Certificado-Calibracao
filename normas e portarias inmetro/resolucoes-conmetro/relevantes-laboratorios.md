# Resoluções Conmetro relevantes para Laboratórios de Calibração

> **Verificar vigência** — resoluções antigas podem ter sido revogadas ou consolidadas. Fonte oficial: https://www.gov.br/inmetro/pt-br/composicao/conmetro

## Resoluções históricas fundamentais

| Resolução | Tema | Relevância para calibração |
|-----------|------|----------------------------|
| Resolução Conmetro nº 11/1988 | Regulamento Geral do Sinmetro | Base institucional — define papéis do Inmetro, Cgcre, IPEMs |
| Resolução Conmetro nº 12/1988 | Quadro Geral de Unidades de Medida | **Obrigatória.** Define unidades aceitas no Brasil (SI) |
| Resolução Conmetro nº 04/2002 | Termos gerais usados em metrologia | Precursor do VIM brasileiro |
| Resolução Conmetro nº 08/2007 | Política de avaliação da conformidade | Diretrizes para OACs |
| Resolução Conmetro nº 05/2008 | Princípios da qualidade no Sinmetro | Diretrizes gerais |
| Resolução Conmetro nº 02/2017 | Programa Brasileiro de Avaliação da Conformidade (PBAC) | Estrutura de RTACs |

## Aplicação no laboratório

### Quadro Geral de Unidades de Medida (Res. 12/1988)
Obrigatório nos certificados de calibração:
- Usar apenas unidades do **SI** e unidades aceitas.
- Não usar unidades obsoletas (kgf, psi, mmHg) como principais — aceitas apenas como unidades de conversão com a unidade SI em destaque.
- Seguir regras de escrita (separação entre valor e unidade, símbolos normatizados).

### Regulamento Geral do Sinmetro
- Define o papel do laboratório no Sistema Nacional.
- Referencia a hierarquia Inmetro → Cgcre → laboratório acreditado.

### Política PBAC
- Referência quando o laboratório atende a mecanismos de avaliação da conformidade (ensaios para certificação).

## Hierarquia normativa

```
     Lei 5.966/1973 (Sinmetro)
              │
              ▼
     Decreto 6.275/2007 (estrutura Inmetro)
              │
              ▼
     Resoluções Conmetro (diretrizes)
              │
              ▼
     Portarias Inmetro (regulamentação)
              │
              ▼
     DOQ-CGCRE / NIT-DICLA (orientação)
              │
              ▼
     Normas ISO/ABNT (técnica)
              │
              ▼
     Procedimentos internos do laboratório
```

## Como anexar uma resolução

1. Baixar o PDF oficial.
2. Salvar como `resolucao-conmetro-<numero>-<ano>.pdf`.
3. Ficha `resolucao-conmetro-<numero>-<ano>.md` com:
   - Data de aprovação e DOU
   - Objeto
   - Vigência
   - Impacto no laboratório
