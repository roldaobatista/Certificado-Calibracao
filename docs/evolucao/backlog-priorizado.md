## Fórmula RICE

```
Reach × Impact × Confidence / Effort
```

- **Reach:** quantos usuários/tenants afetados
- **Impact:** quanto melhora pra cada (1 baixo → 3 alto)
- **Confidence:** quanto temos certeza (0.5 → 1.0)
- **Effort:** semanas-pessoa estimadas
