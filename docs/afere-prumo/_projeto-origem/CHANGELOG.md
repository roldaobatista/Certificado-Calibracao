## Convenções

- **MAJOR.MINOR.PATCH** (Semantic Versioning).
- **MAJOR** muda quando há quebra de compatibilidade.
- **MINOR** muda quando adiciona funcionalidade compatível.
- **PATCH** muda só para correções compatíveis.
- Cada entrada cita o ID do trabalho que originou (`US-MOD-NNN`, `T-MOD-NNN`, `ADR-NNNN`).
- Em release público, criar tag `v<versão>` e copiar a seção correspondente para `gh release create`.
