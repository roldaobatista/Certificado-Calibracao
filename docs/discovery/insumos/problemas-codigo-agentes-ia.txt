1. **Licenciamento e compliance de dependências**

   * Uso de bibliotecas com licença incompatível.
   * Código copiado de exemplos sem checar licença.
   * Dependências com obrigações de atribuição ignoradas.
   * Uso de pacote comercial sem direito de uso.
   * Dependência GPL/AGPL introduzida sem avaliação jurídica.
   * Arquivo `LICENSE` ausente ou incompatível com o produto.

2. **Compatibilidade e versionamento**

   * Breaking changes sem migração.
   * API pública alterada sem versionamento.
   * SDK ou client antigo quebrado por mudança no backend.
   * Contratos GraphQL/REST alterados sem aviso.
   * Mudança de schema sem compatibilidade retroativa.
   * Feature nova quebrando dados antigos.
   * Falta de estratégia blue-green, canary ou rollback.

3. **Migração e evolução de dados**

   * Migration sem plano de rollback.
   * Migration que bloqueia tabela grande em produção.
   * Migration que apaga dados.
   * Migration que assume dados limpos, mas produção tem inconsistências.
   * Backfill sem batch.
   * Backfill sem checkpoint.
   * Backfill sem idempotência.
   * Migration incompatível com múltiplas versões da aplicação rodando ao mesmo tempo.

4. **Integridade financeira**

   * Cálculo monetário usando `float`.
   * Arredondamento inconsistente.
   * Moedas tratadas como número genérico.
   * Falta de controle de casas decimais.
   * Falta de ledger/auditoria em movimentações.
   * Créditos, saldo ou cobrança atualizados sem transação.
   * Reembolso, chargeback ou cancelamento sem fluxo consistente.
   * Operações financeiras sem idempotency key.

5. **Internacionalização e localização**

   * Datas sem timezone explícito.
   * Comparação de datas usando timezone local do servidor.
   * Formatos de data fixos em uma região.
   * Moeda hardcoded.
   * Texto hardcoded em uma língua.
   * Ordenação alfabética incorreta para idiomas diferentes.
   * Falta de suporte a acentos, Unicode e normalização.
   * Bugs com nomes, endereços e documentos internacionais.

6. **Acessibilidade**

   * Botões sem label acessível.
   * Inputs sem `label`.
   * Navegação por teclado quebrada.
   * Contraste insuficiente.
   * Foco visual ausente.
   * Modais sem trap de foco.
   * Erros de formulário não anunciados para leitores de tela.
   * Uso incorreto de ARIA.
   * Componentes clicáveis que não são botões reais.

7. **SEO e metadata**

   * Falta de title/description.
   * Canonical ausente.
   * Páginas indexáveis expondo conteúdo que não deveria.
   * Páginas públicas marcadas como `noindex` por engano.
   * Open Graph incorreto.
   * Sitemap ausente ou desatualizado.
   * Conteúdo renderizado só no client quando precisa ser indexável.
   * Redirecionamentos quebrados.

8. **Cache e invalidação**

   * Cache sem chave por usuário.
