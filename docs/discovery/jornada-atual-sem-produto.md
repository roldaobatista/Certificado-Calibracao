## Por quê este artefato

Pular este passo = construir solução pra problema imaginado. Mapear status quo revela:
- Workarounds aceitáveis (não vale fazer feature pra eles)
- Ferramentas heterogêneas que cliente já paga (oportunidade de integração, não substituição)
- Hábitos profundos que vão resistir a mudança (UX precisa respeitar)
- "Buracos" entre ferramentas onde dor real mora (oportunidade)
